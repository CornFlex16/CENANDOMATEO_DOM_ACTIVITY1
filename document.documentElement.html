document.documentElement is a property in the HTML Document Object Model (DOM) that refers to the root element of an HTML document, which is typically the </html> element. The < html> element acts as the container for all other elements on a web page, including the < head> and < body> elements.

You can use the document.documentElement property to access and manipulate properties of the root < html> element, such as its style and attributes.
